# Hand Tracking MIDI Instrument

## Project Overview
This project is a gesture-controlled virtual musical instrument developed using Python, OpenCV, MediaPipe, Pygame MIDI and Tkinter.

It detects hand movements through webcam and converts gestures into MIDI musical notes and instrument controls.

----

## Features
- Real-time Hand Tracking
- Gesture Controlled Instrument Selection
- Virtual Musical Instrument Playback
- MIDI Sound Generation
- Webcam Based Interaction
- GUI Instrument Selector
- Contactless Music Control
